**1. Project Title / Headline**

📰 Bharat Herald: Business Performance Dashboard (Print vs Digital Analysis)
An interactive Excel dashboard designed to analyze business performance across print and digital channels—focusing on revenue trends, user growth, conversion rates, and ROI to uncover actionable insights.

**2. Short Description / Purpose**

The Bharat Herald Dashboard is an Excel-based analytical tool built to evaluate the company’s transition from print to digital. It helps identify performance trends, inefficiencies, and opportunities for improving revenue, conversion, and cost efficiency.

**3. Tech Stack**

• 📊 Microsoft Excel – Primary platform for dashboard creation
• 📂 Power Query – Data cleaning and transformation
• 🧠 Pivot Tables & Charts – Data aggregation and visualization
• 📈 KPI Cards – Tracking key metrics like revenue, conversion, and growth
• 🎨 Data Visualization – Clean and interactive dashboard design

**4. Data Source**

Source: Simulated dataset provided by Codebasics

The dataset includes business metrics related to print and digital operations such as revenue, users, conversion rates, and costs. It is structured to enable time-based analysis and channel comparison.

**5. Features / Highlights**

**• Business Problem**

As media companies shift from print to digital, understanding performance across channels becomes critical. However, raw data makes it difficult to track revenue trends, user behavior, and ROI efficiently.

**• Goal of the Dashboard**

To build an interactive dashboard that:

Compares print vs digital performance
Tracks key KPIs like revenue, conversion rate, and growth
Identifies inefficiencies and improvement opportunities
Supports data-driven strategic decisions
